# ✨ شرح ميزة العروض والتخفيفات (Offers Feature)

## 📋 نظرة عامة

تم إضافة نظام شامل للعروض والتخفيفات في تطبيق Zamzam ERP يسمح لك بـ:
- ✅ إنشاء عروض ترويجية محددة بفترة زمنية
- ✅ تخصيص المنتجات ضمن كل عرض مع أسعار مخصصة
- ✅ عرض العروض النشطة والقادمة والمنتهية
- ✅ ربط العروض بنظام المبيعات لتطبيق الأسعار المخصصة

---

## 🎯 المميزات الرئيسية

### 1. إنشاء العروض
- إضافة اسم وصف العرض
- تحديد تاريخ البداية والنهاية
- اختيار حالة العرض (نشط/غير نشط)
- إضافة منتجات متعددة مع تحديد السعر المخصص

### 2. إدارة المنتجات في العروض
- اختيار المنتجات من قائمة المنتجات الموجودة
- تحديد السعر المخفض مباشرة
- حساب نسبة التخفيف تلقائياً أو يدوياً
- إضافة/تعديل/حذف المنتجات من العرض

### 3. عرض حالات العروض
- **العروض النشطة**: عروض جارية حالياً
- **العروض القادمة**: عروض مجدولة للمستقبل
- **العروض المنتهية**: عروض انتهت صلاحيتها

## 🗄️ قاعدة البيانات

### جداول جديدة:

#### 1. جدول `Offers`
| العمود | النوع | الوصف |
|--------|------|--------|
| id | INTEGER | معرف فريد |
| name | TEXT | اسم العرض |
| description | TEXT | وصف العرض |
| startDate | DATETIME | تاريخ البداية |
| endDate | DATETIME | تاريخ النهاية |
| status | TEXT | الحالة (Active/Inactive) |
| createdAt | DATETIME | تاريخ الإنشاء |
| updatedAt | DATETIME | تاريخ آخر تعديل |
| createdBy | INTEGER | معرف المستخدم الذي أنشأه |
| isDeleted | BOOLEAN | هل محذوف (soft delete) |

#### 2. جدول `OfferItems`
| العمود | النوع | الوصف |
|--------|------|--------|
| id | INTEGER | معرف فريد |
| offerId | INTEGER | معرف العرض (مرجع) |
| productId | INTEGER | معرف المنتج (مرجع) |
| discountPercent | REAL | نسبة التخفيف |
| discountedPrice | REAL | السعر المخفض |
| quantity | INTEGER | الكمية المتاحة (اختياري) |
| createdAt | DATETIME | تاريخ الإنشاء |

## 📱 كيفية الاستخدام

### إنشاء عرض جديد:

1. من الشاشة الرئيسية، اضغط على **العروض والتخفيفات** من الشريط الجانبي
2. اضغط على زر **عرض جديد** أو الزر العائم
3. ملئ بيانات العرض:
   - **اسم العرض**: مثل "عرض الصيف"
   - **الوصف**: وصف اختياري
   - **تاريخ البداية والنهاية**
   - **الحالة**: نشط/غير نشط
4. اضغط على **إضافة منتج**:
   - اختر المنتج من القائمة
   - أدخل السعر المخفض
   - (اختياري) أدخل نسبة التخفيف
5. كرر الخطوة 4 لإضافة منتجات إضافية
6. اضغط **حفظ العرض**

### تعديل عرض موجود:

1. من قائمة العروض، اختر العرض الذي تريد تعديله
2. اضغط على زر **تعديل**
3. عدّل البيانات المطلوبة
4. اضغط **حفظ العرض**

### حذف عرض:

1. من قائمة العروض، اختر العرض الذي تريد حذفه
2. اضغط على زر **حذف**
3. أكد الحذف

## ⚙️ الخصائص التقنية

### المزودين (Providers):

```dart
// الحصول على جميع العروض
final allOffersProvider = FutureProvider<List<OfferModel>>

// الحصول على العروض النشطة فقط
final activeOffersProvider = FutureProvider<List<OfferModel>>

// الحصول على عرض محدد
final singleOfferProvider = FutureProvider.family<OfferModel?, int>

// الحصول على عرض لمنتج محدد
final productOfferProvider = FutureProvider.family<OfferItemModel?, int>
```

### الحالة (State Management):

يتم استخدام `OffersNotifier` لإدارة عمليات:
- `createOffer()` - إنشاء عرض جديد
- `updateOffer()` - تحديث عرض موجود
- `deleteOffer()` - حذف عرض
